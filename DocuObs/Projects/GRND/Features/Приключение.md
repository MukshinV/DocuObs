## Начать приключение
### Вариант 1
Во время загрузки игры добавить интерактивное окно с надписью  - Adveture
Задача:
- Заинтересовать игрока во время скачивания
- Настроить на игровой процесс
- Сделать несколько оферов
Порядок действий:
1. Появляется кнопка, Start Adventure
2. После нажатия на Start Adventure появляется большой список всех достижений, которые можно получить
	1. Каждое достижение красиво оформлено, с иконкой
	2. У достижения есть награда
	3. Награду можно получить сразу
	4. Некоторые награды можно улучшить
3. Несколько достижений уже выполненно, их можно забрать( это фейк, но...)
