## Функционал первой итерации
1. Игроков переносит в установленную точку
	![[Pasted image 20260420115708.png|574]]![[Pasted image 20260420115742.png|322]]

2. Над машинами начинается отсчет. 10-9-8 ... 3-2-1-Start
3. Игроки должны увидеть одежду друг друга, т.е. они появляются рядом с машинами и когда начинается отсчет, нужно успеть сесть в машину
4. На карте отмечается точка финиша
	1. ![[Pasted image 20260420120110.png|544]]![[Pasted image 20260420120146.png|424]]![[image (23).jpg|967]]
5. Игроки сами нажимают на газ, сами управляют машиной
6. При пересечении линии финиша выводится информация:
	1. Время заезда
	2. Время отрыва
	3. Максимальная скорость
	4. Награда (для премиума награда удваивается)
7. Для заезда инстанс не создается, все гонки происходят на виду у других игроков и происходят постоянно
8. Будет круто, если получится создать небольшую очередь, из 2-3 заездов, чтобы можно было посмотреть другие машины и, например, их Drag Raiting
9. По завершению заезда можно чаще показывать офер с паком двигателей машин и бандлы с машинами

## Вторая итерация
Если есть интерес к Drag Race, механику можно улучшить
1. Убрать у игроков возможность поворота и переделать виджет взаимодействия с авто
	1. Кнопка завести-заглушить машину
	2. Отдельная педаль сцепления
	3. Отдельная кнопка на переключение передачи
	4. Камера фиксируется на машине соперника
	5. Машины едут по сплайну
	6. Сигнал для старта отдает женский НПС
