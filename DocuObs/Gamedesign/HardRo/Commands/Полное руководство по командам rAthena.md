# 📚 Полное руководство по командам rAthena

## 📌 Основная информация

**Конфигурация:** `/conf/atcommand_athena.conf`

- `atcommand_symbol : "@"` — префикс для обычных команд
    
- `charcommand_symbol: "#"` — префикс для GM-команд
    

**Важно:** Некоторые команды недоступны из консоли или скриптов во избежание сбоев.

## 1. СИСТЕМНЫЕ КОМАНДЫ

### 📊 Информация о сервере

|Команда|Описание|
|---|---|
|`@version`|Показывает версию SVN сервера|
|`@rates`|Показывает текущие рейты сервера (опыт, дроп)|
|`@time`|Показывает локальное время сервера с информацией о дне/ночи|
|`@uptime`|Показывает время работы сервера с момента последнего перезапуска|

**Пример вывода `@uptime`:**

text

Server Uptime: 3 days, 8 hours, 6 minutes, 4 seconds.

### 🚫 Баны

|Команда|Описание|
|---|---|
|`@ban <+/- время> <имя>`|Банит аккаунт|
|`@unban <имя>`|Разбанивает аккаунт|
|`@block <имя>`|Блокирует аккаунт|
|`@unblock <имя>`|Разблокирует аккаунт|
|`@charban <+/- время> <имя>`|Банит персонажа|
|`@charunban <имя>`|Разбанивает персонажа|

**Формат времени:**

- `y` / `a` — годы
    
- `m` — месяцы
    
- `d` / `j` — дни
    
- `h` — часы
    
- `mn` — минуты
    
- `s` — секунды
    

**Пример:**

text

@ban +2d Char2
-> банит аккаунт Char2 на 2 дня
